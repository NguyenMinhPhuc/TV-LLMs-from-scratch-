# Glossary / Thuật Ngữ Chuẩn

Song ngữ Anh–Việt cho các thuật ngữ quan trọng dùng trong repository và sách. Mục tiêu: thống nhất cách dịch; giữ nguyên thuật ngữ tiếng Anh bên cạnh tiếng Việt để giảm nhập nhằng. Khi viết tiếng Việt, ưu tiên dạng: *bản dịch* (English term) lần xuất hiện đầu; về sau có thể chỉ dùng tiếng Anh nếu cần chính xác.

| Term                        | Vietnamese                                            | Definition (EN)                                                                                                                 | Định nghĩa (VI)                                                                                                             | Notes                                                                 |
| --------------------------- | ----------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------- |
| LLM (Large Language Model)  | Mô hình ngôn ngữ lớn (LLM)                            | A neural network trained on large corpora to generate and understand text.                                                      | Mạng nơ-ron được huấn luyện trên lượng dữ liệu lớn để sinh và hiểu văn bản.                                                 | Giữ viết tắt LLM nguyên trạng.                                        |
| Reasoning                   | Khả năng lập luận (reasoning)                         | Multi-step logical or mathematical thinking a model performs before final answers.                                              | Quá trình suy nghĩ nhiều bước (logic/toán) mà mô hình thực hiện trước khi đưa ra đáp án cuối.                               | Không dịch thành "suy luận" đơn thuần nếu cần nhấn mạnh tính đa bước. |
| Inference-Time Scaling      | Mở rộng ở thời điểm suy luận (inference-time scaling) | Techniques that improve output quality by increasing computations (e.g. multiple samples) during inference, without retraining. | Các kỹ thuật nâng chất lượng bằng cách tăng phép tính (ví dụ lấy nhiều mẫu) ở giai đoạn suy luận, không cần huấn luyện lại. | Giữ cụm tiếng Anh kèm dịch.                                           |
| Self-Refinement             | Tự tinh chỉnh (self-refinement)                       | Model iteratively critiques and improves its own intermediate outputs.                                                          | Mô hình tự đánh giá và cải thiện các kết quả trung gian qua nhiều vòng.                                                     | Liên quan inference-time scaling.                                     |
| Reinforcement Learning (RL) | Học tăng cường (reinforcement learning)               | Learning via trial-and-error with rewards to optimize policy.                                                                   | Học thông qua thử–sai và phần thưởng để tối ưu chính sách sinh.                                                             | Viết đủ cụm lần đầu.                                                  |
| Distillation                | Chưng cất mô hình (distillation)                      | Transferring knowledge from a larger/ensemble model to a smaller one.                                                           | Truyền tri thức từ mô hình lớn / tập hợp sang mô hình nhỏ hơn.                                                              | Nên giữ từ "distillation".                                            |
| Inference                   | Suy luận (inference)                                  | Running the model to produce outputs without updating weights.                                                                  | Chạy mô hình để tạo kết quả mà không cập nhật trọng số.                                                                     | Phân biệt với training.                                               |
| Token                       | Token                                                 | Smallest input/output unit after model-specific text segmentation.                                                              | Đơn vị nhỏ nhất sau khi phân đoạn văn bản theo tokenizer của mô hình.                                                       | Không dịch sang "mã".                                                 |
| Context Window              | Cửa sổ ngữ cảnh (context window)                      | Maximum number of tokens the model attends to.                                                                                  | Số token tối đa mô hình có thể xử lý cùng lúc.                                                                              | Giữ "context window" trong ngoặc.                                     |
| Batch / Batched Inference   | Batch / Suy luận theo lô                              | Processing multiple inputs together for efficiency.                                                                             | Xử lý nhiều mẫu đầu vào đồng thời để tăng hiệu năng.                                                                        | "Batch" có thể giữ nguyên.                                            |
| Prompt                      | Prompt                                                | Input text (possibly with instructions) given to LLM.                                                                           | Chuỗi văn bản đầu vào (có thể kèm hướng dẫn) đưa cho LLM.                                                                   | Tránh dịch sang "lời nhắc" trong ngữ cảnh kỹ thuật.                   |
| Chain-of-Thought (CoT)      | Chuỗi lập luận (Chain-of-Thought)                     | Explicit reasoning steps generated before final answer.                                                                         | Các bước lập luận rõ ràng mà mô hình sinh ra trước câu trả lời cuối.                                                        | Giữ viết tắt CoT.                                                     |
| Self-Consistency            | Tự nhất quán (self-consistency)                       | Aggregating multiple reasoning samples to pick the most consistent answer.                                                      | Tổng hợp nhiều chuỗi lập luận rồi chọn đáp án nhất quán nhất.                                                               | Xuất hiện trong inference-time scaling.                               |
| MMLU                        | MMLU                                                  | Benchmark covering multi-task language understanding.                                                                           | Bộ đánh giá nhiều nhiệm vụ ngôn ngữ.                                                                                        | Giữ viết tắt gốc.                                                     |
| Leaderboard                 | Bảng xếp hạng (leaderboard)                           | Ordered list of models based on evaluation metrics.                                                                             | Danh sách mô hình được sắp theo chỉ số đánh giá.                                                                            | Dùng trong Appendix F.                                                |
| Elo Rating                  | Elo rating                                            | Pairwise competitive scoring system.                                                                                            | Hệ thống điểm so cặp trong môi trường cạnh tranh.                                                                           | Không dịch "Elo".                                                     |
| Bradley–Terry Model         | Mô hình Bradley–Terry                                 | Probabilistic model for pairwise comparisons.                                                                                   | Mô hình xác suất cho so sánh theo cặp.                                                                                      | Giữ nguyên tên riêng.                                                 |
| LLM-as-a-judge              | LLM-as-a-judge                                        | Using an LLM to evaluate outputs of other models.                                                                               | Dùng một LLM để đánh giá kết quả của các mô hình khác.                                                                      | Không dịch cụm, thêm giải thích.                                      |
| Distillation Dataset        | Tập dữ liệu chưng cất                                 | Curated examples (often with reasoning traces) used to train smaller model.                                                     | Bộ ví dụ (thường kèm bước lập luận) để huấn luyện mô hình nhỏ.                                                              | Xuất hiện trong chương distillation.                                  |
| Reasoning Trace             | Chuỗi lập luận (reasoning trace)                      | The intermediate step-by-step tokens showing thought process.                                                                   | Các token trung gian biểu diễn quá trình suy nghĩ từng bước.                                                                | Đồng nghĩa với chain-of-thought output.                               |

## Guidelines / Hướng dẫn sử dụng
1. Lần xuất hiện đầu tiên trong văn bản tiếng Việt: dùng dạng "dịch (English term)".
2. Thuật ngữ viết tắt (LLM, CoT, MMLU) giữ nguyên, không phiên âm.
3. Tránh dịch các tên riêng, mô hình, tác giả (ví dụ DeepSeek R1, Qwen3, Sebastian Raschka).
4. Nếu một câu chứa quá nhiều tiếng Anh làm khó đọc, có thể ưu tiên tiếng Việt nhưng vẫn giữ từ gốc quan trọng trong ngoặc.
5. Khi cập nhật glossary: thêm hàng mới, giữ cột tiếng Anh nguyên, bổ sung mô tả ngắn gọn.
